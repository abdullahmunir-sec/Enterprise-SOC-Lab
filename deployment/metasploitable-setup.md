# Metasploitable 2 Vulnerable Server

Metasploitable 2 was deployed as an intentionally vulnerable Linux server to act as a realistic target for exploitation and intrusion detection testing.

## Configuration

- OS: Metasploitable 2
- Network: Internal LAN (192.168.1.0/24)
- Role: Vulnerable host used to validate detection of exploitation attempts.

## Purpose

Metasploitable provides numerous exploitable services, allowing the lab to simulate:
- Remote code execution attempts
- Brute-force authentication attacks
- Service enumeration and vulnerability scanning
