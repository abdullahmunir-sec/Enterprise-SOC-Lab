# Windows 11 Endpoint Deployment

A Windows 11 virtual machine was deployed to simulate an enterprise user workstation within the internal LAN.

## Configuration

- Operating System: Windows 11
- Network Segment: Internal LAN (192.168.1.0/24)
- Role: User endpoint used to generate normal and malicious traffic for SOC monitoring.

## Purpose in the Lab

The endpoint is used to:
- Simulate user behavior
- Generate authentication and network activity
- Act as a target for simulated attacks such as brute-force attempts and lateral movement
